## Metadata Standards

### Magazine JSON Fields
- **coverName**: The text/title on the magazine cover
- **date**: Publication date (e.g., "Nov/Dec 1997")
- **description**: Brief description of the issue's content
- **customText**: Additional context, notes, or highlights
- **slug**: URL-friendly identifier (e.g., "nov-dec-1997")
- **coverImage**: Path to local cover image
- **archiveUrl**: Internet Archive detail page URL
